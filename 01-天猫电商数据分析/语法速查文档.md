## 四、sqlalchemy - 数据库连接

### 1. 创建引擎


### 2. 数据写入数据库

```python
# to_sql参数说明：
# - name: 表名
# - con: 数据库连接/引擎
# - if_exists: 'replace'替换, 'append'追加, 'fail'失败
# - index: 是否保存索引列
# - chunksize: 批量写入大小
df.to_sql('table_name', engine, if_exists='replace', index=False, chunksize=1000)
```

## 九、常用语法对照表

| 操作类型 | Pandas语法 | SQL等价 |
|---------|-----------|---------|
| 筛选行 | `df[df['列'] > 值]` | `WHERE 列 > 值` |
| 分组聚合 | `df.groupby('列').agg({'列': 'sum'})` | `GROUP BY 列 SUM(列)` |
| 排序 | `df.sort_values(by='列')` | `ORDER BY 列` |
| 去重 | `df.drop_duplicates()` | `DISTINCT` |
| 连接 | `pd.merge(df1, df2)` | `JOIN` |
| 统计 | `df['列'].mean()` | `AVG(列)` |

## 十、数据类型转换常用方法

| 原始类型 | 目标类型 | 方法 |

| 数值 | 整数 | `s.astype(int)` |
