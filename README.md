# 数据分析项目集

> 北华大学大数据专业数据分析实习项目作品集

## 项目概览

本项目集包含三个完整的数据分析项目，覆盖电商业务分析、机器学习建模、SQL指标体系等核心技能。

---

## 项目列表

### 01-天猫电商数据分析
- **技术栈**: Python、Pandas、NumPy、Matplotlib、MySQL、SQLAlchemy
- **数据规模**: 约5万条商品数据（2024年1月-11月）
- **核心内容**: 数据清洗、特征工程、统计分析、可视化看板
- **GitHub**: 待推送

### 02-二手车价格预测
- **技术栈**: Python、Pandas、Scikit-learn、Matplotlib、Seaborn
- **数据规模**: 约9000条二手车交易数据
- **核心内容**: 数据清洗、特征工程、机器学习建模、模型评估
- **GitHub**: 待推送

### 03-电商业务分析SQL
- **技术栈**: MySQL、SQL（多表关联、聚合函数、CTE、窗口函数）
- **核心内容**: 17项电商业务指标（用户运营、商品管理、销售分析、订单履约、支付管理）
- **GitHub**: 待推送

---

## 技术能力覆盖

| 能力维度 | 项目体现 |
|----------|----------|
| 数据清洗 | Pandas数据清洗、异常值处理、特殊字符处理 |
| 特征工程 | 价格区间、折扣区间、车龄特征、TF-IDF文本特征 |
| 统计分析 | 分组聚合、相关性分析、描述统计 |
| 机器学习 | 线性回归、特征编码、模型评估（R²、RMSE） |
| 可视化 | Matplotlib/Seaborn、Power BI仪表板 |
| SQL技能 | 多表关联、聚合函数、CTE、窗口函数 |
| 业务分析 | 电商指标体系、用户留存、复购率、GMV |
