## 二、项目详情

### 2.1 天猫电商数据分析

**项目定位**：基于天猫平台商品数据，分析品类销售、价格策略、促销效果与补贴ROI

**核心工作**：
- **数据清洗**：处理"万+"格式销量数据，设计向量化清洗函数，数据准确率达98%
- **特征工程**：计算真实折扣率、构建6档价格区间、5档折扣区间等衍生指标
- **统计分析**：品类销售排名、价格带分布、折扣效应相关性分析（相关系数-0.32）
- **数据持久化**：通过SQLAlchemy将5张分析结果表存储至MySQL
- **可视化看板**：生成6维度可视化图表

**关键成果**：
- 识别主力价格带（50-100元占比35%）与TOP10热销品类
- 量化折扣与销量负相关关系，提出最优折扣区间建议
- 输出补贴效率评估报告，筛选高ROI明星商品

**技术栈**：Python、Pandas、NumPy、Matplotlib、MySQL、SQLAlchemy

**文件结构**：
```
01-天猫电商数据分析/
├── analysis.py              # 主分析代码
├── Tools/DF_Tools.py        # 工具模块
├── 天猫数据_1月到11月.xlsx   # 原始数据
├── 分析结果汇总.xlsx         # 分析结果
└── analysis_dashboard.png   # 可视化看板
```

### 2.2 二手车价格预测

**项目定位**：基于二手车交易数据，挖掘价格驱动因素并构建预测模型

**核心工作**：
- **数据清洗**：处理"暂无报价"、单位文本等特殊值，清洗后保留约8900条有效数据
- **特征工程**：计算车龄特征，实现品牌/城市LabelEncoder编码，TF-IDF提取标题文本特征
- **可视化分析**：价格分布对比图、特征相关性热力图、车龄vs价格散点图
- **模型构建**：训练线性回归模型，新能源车R²=0.86，燃油车R²=0.57

**关键成果**：
- 新能源车模型预测准确率达86%，显著优于燃油车
- 发现车龄与售价呈负相关，德系品牌平均售价高于行业均值
- 输出品牌保值率分析报告，为二手车定价策略提供数据依据

**技术栈**：Python、Pandas、NumPy、Scikit-learn、Matplotlib、Seaborn

**文件结构**：
```

├── 代码/Untitled2.ipynb      # Jupyter分析代码

│   ├── 新能源.csv            # 新能源车数据
```

### 2.3 电商业务分析SQL

**项目定位**：构建电商全链路业务指标体系，支持业务决策分析

**核心工作**：
- **用户运营指标**：新增用户数、月活跃用户数、次日留存率、渠道转化率、客单价
- **商品管理指标**：库存周转率、商品毛利润、销售占比、销售分类占比
- **销售分析指标**：GMV、实际销售额、支付转化率、复购率、订单量
- **订单履约指标**：退货率、平均订单时效、订单取消率
- **支付管理指标**：支付方式占比、支付成功率、平均支付时长

**关键成果**：
- 构建覆盖电商全链路的17项核心业务指标体系
- 输出各品类库存周转率与退货率分析报告
- 设计次日留存率与复购率计算逻辑

**技术栈**：MySQL、SQL（多表关联、聚合函数、CTE、窗口函数）

**文件结构**：
```
03-电商业务分析SQL/
├── 1.用户运营指标/   (5个SQL文件)
├── 2.商品管理指标/   (4个SQL文件)
├── 3.销售分析指标/   (5个SQL文件)
├── 4.订单履约指标/   (3个SQL文件)
└── 5.支付管理指标/   (3个SQL文件)
```
