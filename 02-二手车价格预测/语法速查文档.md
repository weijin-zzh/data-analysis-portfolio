## 七、机器学习常用术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 特征 | Feature | 用于预测的输入变量（如车龄、里程） |
| 目标变量 | Target | 要预测的结果（如售价） |
| 训练集 | Training Set | 用于训练模型的数据（通常80%） |
| 测试集 | Test Set | 用于评估模型的数据（通常20%） |
| 过拟合 | Overfitting | 模型在训练集表现好，但在测试集表现差 |
| 数据泄露 | Data Leakage | 训练过程中使用了测试集的信息 |
| 交叉验证 | Cross Validation | 将数据分成K份，轮流作为训练集和测试集 |
| 残差 | Residual | 真实值 - 预测值 |

---

## 八、模型评估指标说明

| 指标 | 公式 | 意义 | 理想值 |
|------|------|------|--------|
| MSE | (Σ(y真实 - y预测)²) / n | 均方误差，衡量预测偏差 | 越小越好 |
| RMSE | √MSE | 均方根误差，与目标变量同量纲 | 越小越好 |
| MAE | (Σ\|y真实 - y预测\|) / n | 平均绝对误差，对异常值不敏感 | 越小越好 |
| R² | 1 - (SS残差 / SS总) | 决定系数，衡量模型解释力 | 越接近1越好 |

## 九、数据预处理流程

```
原始数据 → 数据清洗 → 特征工程 → 数据分割 → 标准化 → 模型训练 → 模型评估
     ↓           ↓           ↓           ↓          ↓          ↓          ↓
 读取CSV    处理缺失值   标签编码    train_test_split  StandardScaler  fit()     predict()
 编码处理   单位转换     TF-IDF提取               transform()     score()
           删除无效数据   特征选择
```
